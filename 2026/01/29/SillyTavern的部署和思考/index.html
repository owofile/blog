SillyTavern的部署和思考

前言

本文主要讲解SillyTavern（俗称酒馆AI）的部署方法和探索更多有趣玩法，文章参考网上一些视频和官方wiki，因为想要帮助小白所以编写的时候会更加详细一些，加上我已经很久没有编写blog了，所以如果出现纰漏或者有问题可以通过邮箱联系交流。

介绍

SillyTavern是基于Node.js构建的轻量级Web，通过前端页面和后端API代理服务的协同运作，实现了各类大语言模型的一站式接入和本地化管理

说人话就是一个API框架，和目前网站上的AI聊天网页不同的是，其内部有一套精密的聊天预设（Prompt）实现更加复杂的对话，这部分挺有意思的，感兴趣可以详细了解一下

这取决于不同AI对于聊天内容的专注度，通过一个巨大的上下文，越靠下的Prompt，会让AI的专注度越高，对于当前对话的影响就会越深，这里先不展开讨论

部署

根据官方 Wiki 的安装指南进行

首先我们需要安装Git，什么是Git？简单来说就是为了方便你拉取SillyTavern这个项目，当然如果你不想下载也可以在浏览器上访问官方的Github手动下载，但为什么我推荐Git，因为在终端，比如Android当你使用Termux进行安装，使用Git只需要一行命令就能安装，但是如果浏览器下载就会异常繁琐

如何安装Git

如果是windows

访问官网：https://git-scm.com/

如果是MacOs

  1. 打开终端并使用以下命令安装 brew：/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. 使用以下命令安装 git：brew install git

如果是Android(Termux)

不论使用什么设备，我最推荐的还是使用Git进行拉取

安装完成后我们就可以进行拉取SillyTavern项目了

第一次使用SillyTavern的顾虑和可能出现的问题

在酒馆中，我们需要先简单了解界面的使用，这里可以去B站搜索教程，很多大佬已经讲述的非常完善，所以不过多赘述，如果只是想要游玩，我们需要注重的只有三个地方，预设，世界书，角色卡，其中角色卡其实代表着一套完整的世界观前端UI和世界书

而我为什么要单独说明世界书，因为世界书除了角色卡外，也可以被单独导入作为工具使用，比如提供状态栏，丰富故事设定，说明等等

而预设则之前讲过，作为提示词中最底下的部分，构成了聊天的文风，破限，思考这些功能，也就是决定了说书人，不同的预设需要根据后续使用的聊天模型看，通过设置预设可以让AI扮演不同的说书人角色，让文章编写更加活灵活现

介绍完简单的功能，首先确保你对于整个页面有简单的认知后可以阅读下述可能让你感到疑惑的问题来得到解答

我想要快速游玩，我应该怎么做？

当你部署完成后，你需要先购买API，拿到API密钥后填写在酒馆里，然后再相关社群比如类脑，寻找对应AI模型的预设，同时需要下载酒馆助手，也就是酒馆中的插件，因为目前大部分大佬写的角色卡都有好看的前端，有了这个插件才能被正确渲染出来，之后下载角色卡导入就可以正常游玩了

我对SillyTavern有更深入的兴趣，想要边玩边了解更多

如果你愿意为此花费更多时间了解，不只是游玩，SillyTavern这个项目本身的很多技术和社区都能满足你，比如你可以加入社群编写角色卡和世界书，这会让你对AI预设词和AI模型本身有更多深入的了解，其本身的逻辑非常有趣，当然你也可以专注于编写前端UI或者预设，这回丰富你的编码技术，抛离SillyTavern有很多有意思或者涩涩的角色卡，其项目的技术和对AI的理解是很领先的，同时这一切在社群中都是免费的，这里也感谢社群中的每个人，正是因为社群中的大佬们不断编写新的角色卡，构建新的玩法，才能有这么多有趣的内容，所以如果你想要进入社群，还请遵循社群的规则，这样才能让社群更好的维护下去

为什么我的SillyTavern和市面上玩到的成品AI酒馆差别这么大？

因为市面上成品的酒馆针对预设和规则等进行的专门的优化和设置，所以一开始的时候可能无法达到那么好的效果，但是多阅读技术文章，或者礼貌询问大佬，一切都会慢慢好起来的，这些是成品酒馆之所以这么贵的原因，他们抓住了小白不愿意琢磨，比较如果是完全不懂的新手，部署整套流程，调教，设置，确实很耗费时间，即使完成后，可能耐心也被消耗了不少，但请记住一点，这里才是一切的起源
