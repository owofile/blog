Dos操作指令

DOS整理

文件路径

[桌面目录cmd.exe] c:\windows\system32\cmd

[xp桌面目录] c:\documents and settings\all users\桌面

[win7桌面目录] c:\users\administrator[name]\desktop\

cmd程序

winver - 检查windows版本

wscript - windows脚本宿主设置

write - 写字板

wiaacmgr - 扫描仪和照相机向导

winchat - xp自在局域网聊天

msconfig.exe - 系统配置实用程序

mspaint - 画图板

mstsc - 远程桌面连接

magnify - 放大镜程序

mmc - 打开控制台

mobsync - 同步命令

dxdiag - 检查DirectX信息

diskmgmt.msc - 磁盘管理实用程序

dcomcnfg - 系统组件服务

dvdplay - DVD播放器

notepad - 打开记事本

narrator - 屏幕讲述人

netstat -an – (TC命令检查接口

sigverif - 文件签名验证系统

shrpubw - 创建共享文件夹

syskey - 系统加密双重加密

services.msc - 本地服务设置

sfc.exe - 系统文件检查器

taskmgr - 任务管理器

eventvwr - 事件查看器

eudcedit - 造字程序

explorer - 打开资源管理器

perfmon.msc - 计算机性能监测程序

regedit.exe - 注册表

rsop.msc - 注册表结果集

chkdsk.exe - 磁盘检查

certmgr.msc - 证书管理实用程序

calc - 启动计算机

charmap - 启动字符映射表

compmgmt.msc - 计算机管理

cleanmgr - 垃圾清理

osk - 打开屏幕键盘

lusrmgr.msc - 本机用户和组

iexpress - mu马捆绑系统自带CAB压缩和自解压工具

fsmgmt.msc - 共享文件夹管理器

utilman - 辅助工具管理器

备忘录

**一般管理员服务器账号****：**administrator

快捷键

[打开输入法] ctrl+空格

运行

explorer.exe [桌面程序]

mstsc [远程桌面程序]

[ip] [访问共享文件夹，多数适用于局域网内的文件运输]

\ [ip]*c$* [默认情况处在同一网络中可以访问c盘]

logoff [注销]

regedit [修改注册表]

services.msc [服务管理窗口]

基础目录控制 [按tab可以补全]

dir [浏览文件] /a=浏览包括隐藏的文件和文件夹

cls [清屏]

ren [重命名]

rd [删除]

cd [进入]

md [创建文件夹]

move [移动]

copy [复制]

xcopy [复制文件夹及文件夹里的文件]

more [显示内容] more 1.txt 2.txt 分页显示多个文档

type [打印]

del [删除文件]

echo nul> 路径 [创建一个自义定内容和路径的文件]

echo. [回响空行]

attrib +s +h [更改属性 s=系统 h=隐藏]

tree [扫描文件]

exit [退出窗口]

命令

arp -a [查看局域网其他人的IP]

shutdown -s -f -t time[关机 s=关机 f=强制 t=时间 time=自义定]

pause [暂停]

color [改变背景和文字颜色]

fsutil file createnew [创建文件，并自定义文件占内存大小]

taskkill /f /im [强制杀死电脑进程]

tasklist [列出进程/管理进程]

nslookup [ip] [检查ip地址的ns服务器]

at [time] [添加计划任务] #如果使用at来打开其他bat有两种方法1: at [time] c:\1.bat 2: at [time] “start c:\1.bat” 加引号是为了把start当做是参数，不过我推荐直接输入路径就好

at [查看计划任务]

telnet towel.blinkenlights.nl [观看ASCLL版星球大战]

[输出] | clip [ |=管道符可以输出字符 clip=剪贴板 管道符快捷键：shift+] [可以联合type 部分系统无法使用 测试：win7可以使用]

&& [同时执行多个命令]

start [运行]

mode con cols=300 lines=80 [改变窗口大小，cols=长 lines=宽]

变量和引用变量与判断

set /p num[num也是可定义字符]=[自义定文字] [定义变量]

%变量名% [引用变量]

：a [创立区间]

goto a [跳转区间]

if — [判断成立后的动作] [判断]

=是赋值

==是等于

用户管理和组管理

whoami /user [查看sid值]

whoami [显示主机名和用户名]

net user [查看所有用户]

net user [name] [passworld] [改密码or创建用户]

net user [name] [passworld] /add [创建一个新用户添加到电脑中]

net localgroup administrators [name] /add [将普通用户移入管理员用户] 提权

net user [name] /del [删除用户]

net user [name] /active:yes [激活禁用用户]

net user [name] /active:no [禁用用户]

组管理

net localgroup [查看所有组]

net localgroup [组name] /add [创建一个组]

net localgroup [组name] [查看一个组的成员]

net localgroup [组name] [用户] /add [将一个用户创建并添加到组里]

net localgroup [组name] [用户] /del [将一个用户从组里删除]

net localgroup [组name] /del [删除一个组]

服务器与端口

net start [服务名] 开启服务/端口

netstat 查询当前使用端口

netstat -an [查询当前端口]

ipconfig [查询本机Ip]

ipconfig /all [查询本机IP和详细信息]

ping [ip] [检查网络流畅] -t [一直ping ctrl+c****停止] -n[修改Ping包的数量]

共享服务

net share [列出对外开放的共享文件夹//包括隐藏共享文件夹$]

net share [共享名] /del [删除共享文件夹，也可以删除共享隐藏文件夹]

注释：服务器重启后会默认重新打开****默认共享

net use [ip]\ipc$ [密码] /nser:[用户名] [连接对方的空链接]

net time [ip] [查询目标主机时间]

copy [路径] [目标ip]\c$\ [复制文件到目标主机c盘上]

at [目标ip] [time] [路径] [给目标主机添加计划任务]

at [目标ip] [查看目标主机时间任务]

at [目标ip] /del [删除目标主机时间任务]

DHCP服务

开始DHCP服务后会自动开启67/68端口

ipconfig /release [释放地址]

ipconfig /renew [重新获取地址]

端口

3389_远程桌面服务(RDP)

445_文件共享服务 [默认微软开机自动打开共享] (SMB)

23_telnet [命令行远程服务]

67/68_DHCP协议端口 [自动分配ip地址服务器]

53_DNS服务监听端口

80_WEB http协议 默认端口

DNS

nslookup [域名/ip] [手动域名解析]

ipconfig /flushdns [清空本地DNS缓存]

ipconfig /displaydns [显示本地DNS缓存]

WEB

当IIS里没有发布网站时，80端口会自动关闭，发布网站后在打开80端口

一些创新

图片隐藏文件.jpg

copy /b 图片.jpg + 需要隐藏的文件.zip 图片.jpg -b表示一个二进位文件

查看

按 ctrl+c 以后输入 cls 清除屏幕

ver 显示当前windows系统的版本号

winver 弹框显示当前windows系统信息

vol 显示当前分区的卷标

label 显示当前分区的卷标，同时提示输入新卷标

label c:system 设置c盘的卷标为system

time 显示或设置当前时间

date 显示或设置当前日期

ver 查看系统详细版本号

msinfo32 查看window详细版本号
